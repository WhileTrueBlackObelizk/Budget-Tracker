# Screenshots

Place your application screenshots here:

- `banner.svg` — README header banner (included)
- `dashboard.png` — Main dashboard view
- `swagger.png` — Swagger API documentation page
- `mobile.png` — Mobile responsive view

**How to take screenshots:**

1. Start the server: `uvicorn backend.main:app --reload`
2. Open `http://127.0.0.1:8000` and add a few sample transactions
3. Take a screenshot of the dashboard → save as `dashboard.png`
4. Open `http://127.0.0.1:8000/docs` → save as `swagger.png`
5. Use browser DevTools (mobile view) → save as `mobile.png`
